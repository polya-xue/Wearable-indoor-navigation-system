The HRM_LowPowerRTC application project comes with an App for Android smartphone:
STM32_BLE_Toolbox.apk.

The STM32_BLE_Toolbox App can be installed on any Android smartphone equipped with
a Bluetooth Low Energy (BLE) chip and Android OS >= ver 4.3.

The HRM_LowPowerRTC application project can be found in this package at:
$BASE_DIR\Projects\STM32L053R8-Nucleo\Applications\Bluetooth_LE\HRM_LowPowerRTC

The STM32_BLE_Toolbox App, once installed, is able show the Heart Rate values in
ppm (pulses per minute) coming from the NucleoL0 board where the HRM_LowPowerRTC
firmware is running.
